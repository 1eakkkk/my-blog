关于本站

技术

基于 Cloudflare Pages + D1 + R2 构建的无服务器论坛。纯静态前端 + Edge Functions 后端。

版本：v2.5.7 · 2026-06-01

Markdown 语法指南

# 标题一级标题（## 二级，### 三级）
**加粗**加粗文字
*斜体*斜体文字
[文字](链接)插入链接
![替代](图片URL)插入图片（上传后自动生成）
`代码`行内代码
```代码块```多行代码块
> 引用引用文字
- 列表项无序列表（数字+点=有序）
| A  | B  |
| --- | --- |
| 1  | 2  |
GFM 表格（需含表头分隔行）
